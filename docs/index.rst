Описание проекта
----------------

Telegram-бот на Python для управления записями клиентов к мастерам. 
Поддерживает гибкое расписание, управление услугами и систему платежей через Telegram Bot Payments.

Основные возможности:

* **Для мастеров**: Управление профилем, услугами, расписанием, получение оплаты
* **Для клиентов**: Просмотр услуг, запись на удобное время, оплата услуг
* **Платежная система**: Интеграция с FreedomPay KG через Telegram Bot Payments
* **Уведомления**: Автоматические уведомления о записях и платежах

Быстрый старт
-------------

1. Установите зависимости:

   .. code-block:: bash

      pip install -r requirements.txt

2. Настройте переменные окружения в файле ``.env``

3. Запустите бота:

   .. code-block:: bash

      python run.py

Подробнее смотрите в разделе :doc:`installation`.

Архитектура
-----------

Проект использует модульную архитектуру:

* ``bot/main.py`` - точка входа и регистрация handlers
* ``bot/handlers/`` - обработчики команд и callback queries
* ``bot/utils/`` - вспомогательные модули
* ``bot/models.py`` - модели данных SQLAlchemy
* ``bot/database.py`` - работа с базой данных

Подробнее смотрите в разделе :doc:`modules`.
